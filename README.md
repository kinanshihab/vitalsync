# VitalSync - Remote Patient Monitoring Dashboard

VitalSync is a real-time health monitoring dashboard designed to help healthcare providers track patient vital signs and receive critical alerts instantly.

![VitalSync Dashboard](https://github.com/kinanshihab/my-website/blob/main/images/projects/vitalsync-full.png?raw=true)

## Features

*   **Real-time Vitals Tracking**: Visual displays for Heart Rate (BPM) and Blood Pressure monitoring.
*   **Critical Alerts System**: Instant notifications for abnormalities like high blood pressure or sensor disconnections.
*   **Patient Status Overview**: Quick-glance list of recent patients with "Stable" or "Attention" status indicators.
*   **Response Dashboard**: Clean, medical-grade UI designed for clarity and rapid decision-making.

## Tech Stack

*   **Vite**: Next Generation Frontend Tooling
*   **TypeScript**: For type-safe, robust code
*   **CSS3**: Custom dashboard grid system and medical styling

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/kinanshihab/vitalsync.git
    cd vitalsync
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  Open your browser at `http://localhost:5173` to view the dashboard.
